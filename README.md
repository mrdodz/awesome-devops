## Installation

1. Open the **Extensions** sidebar in VS Code
2. Search for **Awesome DevOps**
3. Click **Install**
4. Open the **Command Palette** with `Ctrl`+`Shift`+`P` and select **Preferences: Color Theme**
5. Switch to **Awesome DevOps**
6. Enjoy!
